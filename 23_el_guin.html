2.3. El guión

Sirve para que el locutor se guíe o para programar lo que se va a transmitir, la secuencia de comentarios, actividades y temas que incluirá el programa. También se le denomina escaleta y consta de dos partes:

  • Técnica: refleja el tiempo, orden de entrada de los participantes, cabecera e instrucciones.
  • Literaria: recoge el texto de las locuciones.

Estructura del guion

  • Cabecera: Es la parte inicial donde se muestran los datos relacionados con el título, el autor, la fecha y hora de emisión, y cualquier otro detalle que permita la fácil localización.
  • Sintonía: Secuencia breve de notas musicales, tras la cual suele facilitarse el título del programa o el nombre de la emisora o de aquello que se pretende identificar. 
  • Indicativo: Intervención muy breve (sólo títulos, nombres) que recuerda al oyente el programa o la emisora que está escuchando, o ambas cosas. Puede incluir sintonías o ráfagas musicales o no incluirlas. 
  • Entradilla: Intervención breve en la que el locutor centra el programa (o la sección) que va a venir a continuación.
  • Secciones: Cada una de las partes diferenciadas en las que se puede dividir un programa.
  • Cuñas: Son montajes sonoros cortos (treinta segundos sería una media), dramatizados o no, que se utilizan como promoción (sirven para dar publicidad, para anunciar.
  • Cortinilla: Ráfaga de separación (habitualmente entre secciones), casi siempre pregrabada.
